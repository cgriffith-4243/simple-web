CGRIFFITH ART WEBSITE

#Description

This is a static html website. This website is an artist portfolio that showcases artwork I (Cameron Griffith) have done over the past decade.

The header of this website are the title, "CGriffith Art", and the navigation bar. The user will be able to navigate the website to read artist information and view the artworks, organized by art medium. This website consists of five pages: Home, About, Drawings, Paintings, Contact. The footer contains copyright information for the website.

Home page consists only of an example image and is meant as a launchpad for the user's experience on the website.

About page contains a short description/bio and an example list of mediums used for the artwork. Two images showcasing an artwork are also on the page.

Drawings and Paintings pages consist of images of the artworks, divided respectively by medium. A table organizes the images and uses rowspan/colspan for larger images. The user can click on a portfolio image and they will be redirected to the full size version of the image.

Contact page contains the artists social media and contact information. The artist instagram and email are linked. The page also includes a form where the user can submit their name, email, and fill in a textfield for a message they would like to send.
